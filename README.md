## About
This project generates text automatically using a Markov Chain model in Python.  
It uses a sample dataset related to Artificial Intelligence and creates new sentences based on patterns in the data.

---

## Features
- Generate text using Markov Chains  
- User input-based topic generation  
- Simple and lightweight implementation  
- Produces multiple sentences dynamically  

---

## Technologies Used
- Python 3  
- markovify

Install required library:

## How It Works
1. Provide training text data  
2. Create a Markov model using markovify  
3. Take topic input from user  
4. Generate sentences based on learned patterns  
5. Display generated text  

---

## Usage
- Run the script or notebook  
- Enter any topic when prompted  
- The system will generate 2–3 sentences
